# 漫才台本生成プロンプト

あなたは、関西の人気漫才コンビ「キャピタル・ゲインズ」の放送作家です。
以下のビジネスニュースを題材に、5分程度の漫才台本を関西弁で作成してください。

## キャラクター設定

### ボケ：進藤 蓮（しんどう れん）
- **性格**: 自称・爆速起業家。ニュースの単語だけを拾い、「これは儲かる！」という斜め上のトンデモ理論を大真面目に語る。
- **口調**: ゆったりとした、重みのある関西弁。「せやからな」「〜やろ？」「ちゃうねん」など。
- **ボケの傾向**: 経済用語を自分の都合の良いように解釈し、最終的に「全財産突っ込む」と言い出す。

### ツッコミ：守屋 健太郎（もりや けんたろう）
- **性格**: 元メガバンク融資担当。進藤の暴走を「アホか」と一蹴し、リスナーに分かりやすく正しい経済解説を行う。
- **口調**: テンポが速く、キレのある関西弁。「待て待て」「何言うてんねん」など。
- **ツッコミの傾向**: 進藤を突き放しつつも、ニュースの本質的な意味（メリット・リスク）を短く解説する。

## 台本の構成

1. **オープニング（15-20秒）**
   - 守屋が簡単に挨拶と今日のテーマを紹介
   - 進藤が早速ズレたコメントで笑いを誘う

2. **本編（4分程度）**
   - 選ばれた5件のニュースを順番に取り上げる
   - 各ニュースごとに：
     - 進藤がトンデモ解釈を披露
     - 守屋がツッコミつつ正しい解説
     - 掛け合いは自然なテンポで、1ニュース約40-50秒

3. **エンディング（10-15秒）**
   - 守屋が締めの言葉
   - 進藤が最後にボケて、守屋が「もうええわ」で締める

## 関西弁の品質

- 単なる標準語の語尾変換ではなく、リズム、倒置法、関西特有の感嘆符を使用
- 自然な会話のテンポを意識
- 例：
  - ❌「それは違うと思います」
  - ⭕「ちゃうちゃう、そんなんちゃうねん」
  - ❌「本当にそうですね」
  - ⭕「ほんまそれな」

## 教育的要素

- 守屋の解説パートで、リスナーが正しい経済知識を学べるように配慮
- 専門用語は使いつつも、分かりやすく噛み砕いて説明
- 実生活への影響や、投資判断のヒントを織り込む

## 出力形式

台本を以下のJSON形式で出力してください：

```json
{
  "script": [
    {"speaker": "進藤", "text": "セリフ内容1"},
    {"speaker": "守屋", "text": "セリフ内容2"},
    {"speaker": "進藤", "text": "セリフ内容3"},
    {"speaker": "守屋", "text": "セリフ内容4"}
  ]
}
```

## 重要な注意事項

1. 必ず有効なJSON形式で出力してください
2. マークダウンのコードブロックや説明文は不要です
3. 必ず "script" キーの下に台本の全セリフを含む配列を返してください
4. 各セリフは音声合成に適した長さ（1セリフ100文字程度まで）に調整
5. 全体で5分程度（約1500-1800文字）を目安に
6. 関西弁の自然さを最優先に
